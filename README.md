# SoftDeskAPI

## Description

A simple API for issue tracking system applications.

## Installation

### Prerequisites

- `python` >= 3.8, `pip`, `pipenv`

### Get started

```bash
pipenv install
```

```bash
pipenv shell
```

## Running the app

```bash
python manage.py runserver
```
