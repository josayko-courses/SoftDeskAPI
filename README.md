# SoftDeskAPI

## Description

A simple API for issue tracking system applications.

## Installation

### Prerequisites

- `python` >= 3.9, `pip`, `pipenv`
- If `python` < 3.9, manage different `python` versions with **[pyenv](https://github.com/pyenv/pyenv)**

### Get started

```bash
pipenv install
```

```bash
pipenv shell
```

## Running the app

```bash
python manage.py runserver
```
