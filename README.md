# SoftDeskAPI

## Description

A simple API for issue tracking system applications.

## Installation

### Prerequisites

- `python` >= 3.6, `pipenv`

Create a virtual environment with `pipenv`:

```bash
# install pipenv with pip
pip install pipenv
```

```bash
pipenv shell
```

### Get started

```bash
pipenv install
```

## Running the app

```bash
python manage.py runserver
```
